💡 使用提示 / Tips

关键词 Keywords: 多个触发词用英文逗号分隔，不区分大小写，支持中英文。
Separate multiple triggers with commas. Case-insensitive. Chinese & English supported.

语音 URL: 填写语音文件的公开 URL（支持 .ogg, .mp3）。填写后机器人将发送语音而非文字。
Enter a public URL to an audio file (.ogg, .mp3). When set, the bot sends this voice clip instead of text.

排序: 表格中越靠前的关键词优先级越高，第一个匹配即触发。
Rows higher in the table have higher priority — the first match triggers.
